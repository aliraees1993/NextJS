# Discover Cafes App

## This app includes

-   Unsplash API
-   SWR
-   Airtable as storage
-   Geolocation API
-   Server Side Rendering
-   Client Side Rendering
-   Serverless APIs

To install node modules: `yarn install`

To run: `yarn start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
